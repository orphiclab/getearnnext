# GetEarnNext

[![Next.js](https://img.shields.io/badge/Next.js-15.0-black?style=flat&logo=next.js)](https://nextjs.org/)
[![Tailwind CSS](https://img.shields.io/badge/Tailwind_CSS-3.4-38B2AC?style=flat&logo=tailwind-css)](https://tailwindcss.com/)
[![Framer Motion](https://img.shields.io/badge/Framer_Motion-12.4-black?style=flat&logo=framer)](https://www.framer.com/motion/)

A high-converting, modern SaaS landing page designed to help users monetize their digital presence. Built with Next.js, Tailwind CSS, and Framer Motion.

## 🚀 Features

- **Modern Glassmorphism UI**: Deep black background (`#000103`) paired with vibrant mint green accents (`#4ADE80`).
- **Shiny Call-To-Actions**: Highly engaging, light-sweeping animated primary buttons.
- **Scroll Animations**: Smooth entrance animations for all sections powered by Framer Motion.
- **Responsive Design**: Flawless mobile-first architecture utilizing Tailwind CSS.
- **Dynamic Backgrounds**: Floating particles and subtle glowing ambient blobs mapped across the viewport.

## 💻 Tech Stack

- **Framework**: [Next.js](https://nextjs.org) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Typography**: Google Fonts (Inter & Playfair Display)

## 🛠️ Getting Started

### Prerequisites

Ensure you have Node.js 18.x or later installed on your machine.

### Installation

1. Clone the repository
```bash
git clone https://github.com/orphiclab/getearnnext.git
```

2. Navigate into the project directory
```bash
cd getearnnext
```

3. Install dependencies
```bash
npm install
```

4. Start the development server
```bash
npm run dev
```

5. Open [http://localhost:3000](http://localhost:3000) in your browser to view the application.

## 🎨 Design System

The platform utilizes the **GetEarnNext** design language:
- **Primary Background**: `#000103`
- **Primary Accent (Green)**: `#4ADE80`
- **Surface Panels**: `rgba(255, 255, 255, 0.03)`
- **Surface Borders**: `rgba(255, 255, 255, 0.08)`
